# Project AsciiGame

__A puzzle game made in C using the terminal__

## Requirements

* GCC
* ncurses

Linux - ``sudo apt-get install libncurses5-dev``
Mac OS - ``brew install homebrew/dupes/ncurses``

## Build

``gcc -pedantic-errors -pedantic -Wall asciiGame.c -o asciiGame -lncurses ``

## Running

``./asciiGame
